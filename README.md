# Serjo Studio Portfolio

A premium, high-performance portfolio for Serjo Studio, architecting intelligent digital systems.

## Features

- **Alternating Scroll Experience**: A unique blend of vertical and horizontal scrolling for a dynamic user journey.
- **AI-Driven Design**: Focused on AI automation, MCP agents, and high-performance web architecture.
- **Premium Aesthetics**: Dark mode, glassmorphism, fluid animations (Framer Motion), and custom cursor.
- **Interactive AI Chatbot**: A floating AI assistant to help users navigate and learn about services.
- **Responsive & Optimized**: Built with React, Vite, and Tailwind CSS for maximum speed and responsiveness.

## Sections

1.  **Hero**: Impactful introduction with parallax effects.
2.  **Services**: Overview of core expertise in web architecture and AI.
3.  **Process**: Step-by-step methodology from concept to autonomy.
4.  **Projects**: Selected works showcasing engineered solutions.
5.  **About**: Core philosophy and performance metrics.
6.  **Testimonials**: Client validation and feedback.
7.  **Pricing**: Transparent investment models.
8.  **Team**: Meet the architects behind the systems.
9.  **Tech Stack**: Detailed look at the technical ecosystem.
10. **Contact**: Minimalist, blueprint-inspired contact form.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **AI Integration**: MCP (Model Context Protocol) concepts

## Getting Started

1.  Install dependencies: `npm install`
2.  Run development server: `npm run dev`
3.  Build for production: `npm run build`
